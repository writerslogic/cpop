# Technical Specifications

This directory contains formal specifications for the **Proof-of-Process (PoP)** protocol suite.

## Documents

- **[Evidence Format](evidence-format.md)**: Packet structure and serialization (WPkt/JSON).
- **[Process Declaration](process-declaration.md)**: Signed author attestation format.
- **[Ratchet Key Hierarchy](ratchet-key-hierarchy.md)**: Key derivation and rotation protocol.
- **[Architectural Hardening](architectural-hardening.md)**: Tier 4 protections (Memory Locking, Anti-Debug).
- **[Behavioral Metrics](behavioral-metrics.md)**: Keystroke dynamics and forensic analysis.
- **[Persistence & Fault Tolerance](persistence-fault-tolerance.md)**: WAL, WAR, and crash recovery.

## Status

All specifications are currently in **Draft** status and follow the **`draft-condrey-rats-pop`** IETF standard.
