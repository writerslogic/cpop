# JSON Schemas

Formal data models for WritersLogic objects.

## Schemas

- **[Evidence V1](evidence-v1.json)**: The core evidence packet format.
- **[Declaration V1](declaration-v1.json)**: Process attestation format.
- **[WAR Block V1](war-block-v1.json)**: ASCII-armored record format.

## Usage

These schemas can be used for:
1. Automated verification of exported packets.
2. Cross-platform implementation of WritersLogic clients.
3. Documentation of the data model.
